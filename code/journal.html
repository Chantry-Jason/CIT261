I learned there are many different types of instantiation. I coded two types, prototypal and pseudo classical. It took me awhile reading up on these in order to understand them. I first copied some code to step through to gain a better understanding of what was happening for each type. I was finally able to write my own code for both. I can see the value in using instantiation to assign user data to objects quickly. These then can be recorded to disk or a database faster than separate variables or arrays.

I had difficulty with assigning stylings to that variable until I looked up and remembered that the function returns an array of all objects it can find. In my case there is only one, but that meant I needed to loop through the array until I reached the end. I couldn't simply say:

variableWithClassName.style.bordercolor = "forestgreen";

I needed to create a for loop:
