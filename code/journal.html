Started working on topic 2 which Is to create object functions, properties, methods and instantiation. I did not remember learning this in my previous courses. So, I learned what I could from w3Schools, Daily JS, and my brother Scott who is a JS programmer.

I learned there are many different types of instantiation. I coded two types, prototypal and pseudo classical. It took me awhile reading up on these in order to understand them. I first copied some code to step through to gain a better understanding of what was happening for each type. I was finally able to write my own code for both. I am not an expert at this yet, but I feel I got a pretty good basic understanding. I will use this more as the course goes on to gain a working knowledge of the code.

I definitely like the pseudo classical approach much better. It is easier to understand and code. It is fairly quick to program the function, objects, properties, and methods.

I can see the value in using instantiation to assign user data to objects quickly. These then can be recorded to disk or a database faster than separate variables or arrays.

May 27, 2018

submitted topic #7, manipulating css using javascript. This was a fun exercise that got me thinking again about what I had learned several semesters ago. I used getElementsByClassName to assign the class object to a variable. I had difficulty with assigning stylings to that variable until I looked up and remembered that the function returns an array of all objects it can find. In my case there is only one, but that meant I needed to loop through the array until I reached the end. I couldn't simply say:

variableWithClassName.style.bordercolor = "forestgreen";

I needed to create a for loop:

for(i=0; i<classToMod.length; i++) {classToMod[i].style.borderColor = "forestgreen";}</p>

this would loop through all instances of the class name and assign the styling to all of them. Again, in my case it was only one so i could have simply said classToMod[1].style.borderColor = "forestgreen"; but decided to loop. The loop method allows me to add the styling inline which will override the styling I specified in my general style section.
